# Installing the SFMC Node.js SDK

## Overview

OGDP communicates with Teledyne's Slocum Fleet Mission Controller (SFMC)
using the official Node.js SDK supplied with SFMC.

The SDK is **not open source** and is therefore not included with OGDP.

## Obtaining the SDK

Install SFMC using the official Teledyne installation packages.

The SDK is included in the SFMC Toolbox package.

## Required file

sfmc.tgz

## Installing

Example:

npm install /path/to/sfmc.tgz
